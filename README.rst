Как настроить рабочее окружение?
--------------------------------

Требуются следующие установленные зависимости:

* `virtualenv`
* GNU Make `make`
* Python 3.6 `python3.6`


Для того, чтобы настроить рабочее окружение, нужно просто вызвать
цель `develop` из `Makefile`

.. code-block::
   make develop
